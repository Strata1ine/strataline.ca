## Core Problem
Site schema targets "GeneralContractor" but the brand's true differentiator is "dust-free" process. Keywords in business.json are just city names - no service terms. SEO meta on service pages is descriptive prose, not keyword-optimized. "Dust free renovation" as a search term returns cleaning companies, not Strataline - the term has low search intent alignment.

## Top Competitors by Service
- **Popcorn removal**: ceilingspecialists.ca, masterceiling.ca, prestigepaintinggta.ca, rollthewall.ca, epfproservices.com, timrenovation.ca
- **Stairs**: stairsteps.ca, titanfloors.ca, qualityhardwoodrefinish.com, durhamstairs.com, focusonflooring.ca
- **Kitchen/Bath**: sosna.ca, smartreno4u.ca, easyrenovation.ca, aandrrenovations.com, pabbathrooms.com, gmco.ca
- **General reno**: csgrenovation.ca, renovatingforyou.com, renotec.ca

## Key Gaps Identified
1. No cost/pricing content pages (competitors rank for "popcorn ceiling removal cost Toronto", "kitchen renovation cost 2026")
2. No location-specific landing pages despite serving 30+ cities
3. SEO meta fields contain prose descriptions instead of keyword strings
4. No blog/guide content for informational queries
5. "Dust-free" is not a commonly searched modifier - users search by service + location
6. Competitors use "[service] + [city]" as primary keyword pattern
7. No FAQ schema content on service pages

## Target Demographic
- Homeowners 35-60, household income $100K+, owning detached/semi in GTA suburbs
- Renovation-ready: homes built 1970s-2000s needing updates (popcorn ceilings, carpet stairs)
- Values: cleanliness, professionalism, minimal disruption, quality finishes
- Search behavior: "[service] [city] cost", "[service] near me", "best [service] contractor [city]"
